Header file with macros that print various messages (usually, the content of various F\+L\+I\+NT variables) when the program is compiled with -\/D\+D\+E\+B\+UG=lvl option, where lvl indicates the level of detail (0, 1, 2, ... etc). By default, these macros are disabled to keep the code as fast as possible.
